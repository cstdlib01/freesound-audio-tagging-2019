# Kaggle Freesound Audio Tagging 2019 Competition Solution

This is a solution for [Kaggle Freesound Audio Tagging 2019 Competition](https://www.kaggle.com/c/freesound-audio-tagging-2019/):

- Multi label classigication of audible raw sound data source.
- Two different training dataset; clean curated set and noisy labeled set.

This solution solves by following phases:

- Preprocessing: Convert raw audio samples (1D) into mel-spectrogram 3 channel image like format (2D).
- Model: Multi label CNN.
